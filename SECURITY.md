# Security Policy

## Scope

The AI/ML Compendium is a **curated resource list** — it contains links to external content, not executable code or services. There is no deployed application, API, or user-facing service to exploit.

## Reporting a Vulnerability

If you discover a security concern (e.g., a link pointing to a malicious site, a phishing URL, or compromised content), please report it by:

1. **Opening a [Broken Link issue](https://github.com/dsai-iitbhilai/the-ai-ml-compendium/issues/new?template=broken-link.yml)** with severity "Resource is gone with no replacement"
2. **Or emailing the maintainers** at the email listed on the organization profile

We take malicious link reports seriously and will remove the resource within 48 hours of confirmation.
