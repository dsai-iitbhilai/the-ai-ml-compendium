## Key Concepts Checklist

- [ ] Tufte's principles: data-ink ratio, chart junk, lie factor
- [ ] Choosing the right chart type for your data and question
- [ ] matplotlib figure and axes API, subplots, custom styles
- [ ] seaborn: `relplot`, `displot`, `catplot`, `lmplot`
- [ ] Plotly Express for interactive figures
- [ ] Color theory: sequential, diverging, categorical palettes
- [ ] Statistical plots: boxplot, violin plot, ECDF, Q-Q plot
- [ ] Histograms vs KDE — when to use each
- [ ] Pair plots and correlation heatmaps
- [ ] Time series visualization: line plots, rolling windows, seasonality
- [ ] Dashboard construction with Plotly Dash or Streamlit
