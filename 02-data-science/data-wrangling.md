# Data Wrangling

> Transforming raw, messy data into clean, reliable, and analysis-ready datasets. Covers pandas, NumPy, missing data, reshaping, joins, feature preparation, and reproducible data pipelines.

**Last Reviewed:** 2026-06

**Prerequisites:** [Python for AI](../01-foundations/programming/python-for-ai.md)

---

> 💡 **Tip:** Spend more time mastering data wrangling than learning new models. In real-world projects, data cleaning often takes 70–80% of the effort.
