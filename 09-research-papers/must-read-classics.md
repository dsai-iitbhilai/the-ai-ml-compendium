# Must-Read Classics

> Foundational papers that shaped modern machine learning and deep learning.

## Recommended Reading Order for Beginners

1. **Random Forests** — accessible ensemble method
2. **Dropout** — simple idea with big impact
3. **AlexNet** — see how deep learning solved a hard problem
4. **ResNet** — understand the innovation that unlocked depth
5. **Attention Is All You Need** — the modern foundation
