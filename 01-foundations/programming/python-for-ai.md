## Key Concepts Checklist

- [ ] NumPy arrays vs. Python lists (broadcasting, slicing, reshaping)
- [ ] Pandas DataFrames (indexing, groupby, merge, apply)
- [ ] Matplotlib + seaborn (line, scatter, histogram, subplots)
- [ ] Virtual environments (venv, conda) and package management (pip)
- [ ] Jupyter notebooks (markdown, magic commands, kernels)
- [ ] scikit‑learn API (fit / predict / transform, pipelines)
- [ ] Basic data wrangling: handle missing values, encode categories, scale features
- [ ] Writing modular, reusable Python code (functions, classes, modules)
- [ ] Debugging (pdb, logging, assertions)
- [ ] Basic performance profiling (timeit, %timeit)
