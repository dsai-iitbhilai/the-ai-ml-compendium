# Data Scientist Learning Path

> For analysts, statisticians, or anyone with **data manipulation skills** looking to move into ML-focused data science.

## Path Overview

Focus on statistical modeling, experimentation, and deriving insights from data. Less emphasis on infrastructure.

## Core Modules

| Step | Module | Key Resources | Goal |
|---|---|---|---|
| 1 | [Module 01 — ML Fundamentals](../01-fundamentals/) | Metrics, evaluation, cross-validation | Solid evaluation practice |
| 2 | [Module 02 — ML Algorithms](../02-machine-learning/) | Linear/Logistic Regression, Trees, Ensembles | Interpretable models |
| 3 | [Module 04 — NLP](../04-nlp/) | Text processing, sentiment, topic modeling | Unstructured data analysis |
| 4 | [Module 06 — Advanced Topics](../06-advanced-topics/) | XGBoost, feature engineering, hyperopt | High-performance modeling |

## Elective Modules

| Module | When to Take |
|---|---|

## Additional Focus Areas

- **A/B Testing** — statistical significance, multiple testing correction
- **Experiment Design** — randomization, sample size calculation
- **Causal Inference** — propensity scores, instrumental variables
- **Data Storytelling** — dashboards, executive summaries
