# Learning Paths

> Structured sequences through the compendium tailored to different goals and backgrounds. Each path links to specific modules, resources, and projects.

## Available Paths

| Path | Target Audience | Est. Time | Starting Module |
|---|---|---|---|
| [Absolute Beginner](./absolute-beginner.md) | No prior ML experience, basic Python | 6–12 months | 01-Fundamentals |
| [Data Scientist](./data-scientist-track.md) | Analysts moving into ML | 4–8 months | 01-Fundamentals |
| [ML Engineer](./ml-engineer-track.md) | Builders focusing on deployment and scale | 3–6 months | 03-DL / 07-MLOps |
| [GenAI Engineer](./genai-engineer-track.md) | Building with LLMs, RAG, and generative models | 3–6 months | 04-NLP / 05-GenAI |
| [Agentic AI](./agentic-ai-track.md) | Creating multi-agent systems and tools | 2–4 months | 05-GenAI / 08-Projects |

## How to Use a Path

1. Each path includes **core modules** (required) and **elective modules** (recommended)
2. Within each module, follow the **Resource Table** entries marked with your level
3. Complete the corresponding **Projects** to solidify learning
4. Revisit the **Interview Prep** module when job-ready
