## Types of Coding Rounds

| Type | Description | Example |
|---|---|---|
| **Algorithm** | Classic DSA (arrays, trees, graphs, DP) | "Two Sum", "LRU Cache" |
| **ML from Scratch** | Implement a model without libraries | "Logistic Regression using numpy" |
| **ML with Libraries** | Use sklearn/PyTorch to solve a task | "Train a classifier on this dataset" |
| **Data Manipulation** | pandas, numpy, data cleaning tasks | "Clean this CSV and compute features" |
| **Take-Home Project** | Full ML pipeline in ~4-8 hours | "Build and deploy a sentiment classifier" |

## Recommended Preparation Path

1. **Data Structures & Algorithms** — Arrays, HashMaps, Trees, Graphs, DP (NeetCode 150)
2. **Python Proficiency** — List/dict comprehensions, generators, `*args`/`**kwargs`, decorators
3. **ML Implementation** — Linear/Logistic Regression, KNN, K-Means, Decision Tree in numpy
4. **Library Familiarity** — pandas groupby/merge, sklearn pipelines, PyTorch Dataset/DataLoader
5. **Mock Interviews** — Pramp, interviewing.io, or peer practice

| Pattern | Frequency | Example |
