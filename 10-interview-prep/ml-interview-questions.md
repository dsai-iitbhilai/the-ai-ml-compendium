## Topic Checklist

| Topic | Must-Know |
|---|---|
| **Supervised Learning** | Bias-variance tradeoff, cross-validation, regularization (L1/L2), SVM kernels, tree-based methods |
| **Unsupervised Learning** | K-means, DBSCAN, PCA, t-SNE, GMM |
| **Deep Learning** | Backpropagation, activation functions, batch norm, optimizers (Adam, SGD), dropout |
| **NLP** | Word embeddings, Transformers, attention, BERT, GPT |
| **CV** | CNNs, pooling, data augmentation, transfer learning, YOLO |
| **Evaluation** | Precision/recall, AUC-ROC, F1, confusion matrix, RMSE, MAE |
| **MLOps** | CI/CD, model serving, A/B testing, drift detection |
| **Math** | Linear algebra (eigendecomposition, SVD), probability (Bayes, distributions), calculus (gradients, chain rule) |

## Sample Questions

| Question | Topic |
|---|---|
| Explain the bias-variance tradeoff. How do you diagnose each? | Fundamentals |
| What's the difference between L1 and L2 regularization? When would you use each? | Linear Models |
| How does dropout work and why does it prevent overfitting? | Deep Learning |
| Walk through how backpropagation computes gradients. | Deep Learning |
| How would you detect data drift in a production model? | MLOps |
| Compare Bagging vs. Boosting. Give examples of each. | Ensemble Methods |
| Explain the attention mechanism in Transformers. | NLP |
