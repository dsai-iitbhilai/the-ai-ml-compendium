# 08 — Projects & Examples

> **Goal:** Apply knowledge from previous modules by building real-world ML projects. Projects are organized by difficulty level and domain.

## Structure

| Directory | Focus | Target Audience |
|---|---|---|
| [Beginner](./beginner/) | Small, focused projects with guided steps | Entry-level learners |
| [Intermediate](./intermediate/) | Multi-component projects with real datasets | Learners with fundamentals |
| [Advanced](./advanced/) | Research-level, distributed, or production-grade systems | Experienced practitioners |
| [End-to-End Systems](./end-to-end-systems/) | Full-stack ML systems (model + API + UI + infra) | All levels (scoped) |

## Project Philosophy

- **Start small, iterate fast** — each project can be extended
- **Reproducibility** — every project includes setup instructions and data sources
- **Portfolio-ready** — projects are designed to showcase on GitHub/resume
- **Tool agnostic** — choose your own framework (PyTorch, TF, JAX)

## Suggested Workflow

1. Pick a project at or slightly below your comfort level
2. Fork the template and follow the README
3. Extend with your own ideas
4. Deploy (optional) and share
