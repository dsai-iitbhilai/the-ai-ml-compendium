# Beginner Projects

> Hands-on projects designed for learners who know Python basics and are starting ML.

## Project List

| Project | Description | Skills Practiced | Dataset |
|---|---|---|---|
| **Titanic Survival Predictor** | Binary classification with feature engineering | pandas, Logistic Regression, Random Forest | Titanic (Kaggle) |
| **Spam vs. Ham Classifier** | Text classification with NLP basics | TF-IDF, Naive Bayes, evaluation metrics | SMS Spam Collection |
| **House Price Predictor** | Regression on Boston-style housing data | Linear Regression, feature scaling, RMSE | California Housing |
| **Iris Flower Classifier** | Classic multi-class classification | KNN, Decision Trees, confusion matrix | Iris (UCI) |
| **MNIST Digit Recognizer** | Image classification with neural nets | PyTorch/TF basics, CNN, softmax | MNIST |

## Getting Started

1. Set up a virtual environment and install `scikit-learn`, `pandas`, `matplotlib`
2. Choose one project and follow the linked notebook
3. Experiment with at least two different models and compare results
4. Write a short summary of findings
